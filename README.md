# Better Hindu Party

A satirical, single-page political-comedy site: *"Dharma over deal-making."*

The Better Hindu Party (BHP) is not a real party. The bit holds the establishment's own
stated values at their word — to a standard so literal and uncompromising that every deal,
defection, and exam leak looks exactly like the *Adharma* it claims to oppose.

## Contents

- `index.html` — the entire site (manifesto, membership steps, and a mock suggestion-form
  prop). No build step, no dependencies, no backend. The form submits nowhere; nothing is
  stored or sent.

## Running locally

Open `index.html` directly in a browser, or serve the directory:

```sh
python3 -m http.server 8000
```

then visit `http://localhost:8000`.

## Disclaimer

This is a work of satire. The BHP is not a registered party, takes no donations, fields no
candidates, and has no affiliation with any real organisation.
